# Litify — D2C Room Décor E-Commerce Platform

## Overview
Litify is a full-stack direct-to-consumer e-commerce platform for room décor,
including posters, tapestries, rugs, and lights. Aniruddha is co-founder and
product lead. The store is live at https://litify.shop/.

## Tech Stack
- WordPress and Elementor Pro as the content and storefront foundation
- Custom frontend components for catalog, cart, and checkout flows
- Razorpay for payment integration
- Standard analytics and marketing tooling for go-to-market
